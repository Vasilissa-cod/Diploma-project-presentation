# Скриншоты для приложения

Добавьте скриншоты тестирования в эту папку с следующими именами:

| Файл | Описание |
|------|----------|
| `01-dashboard.png` | Главная страница открытого аккаунта (Dashboard авторизованного пользователя) |
| `02-homepage.png` | Главная страница сайта (Лендинг FatSecret) |
| `03-kanban-miro.png` | Kanban-доска в Miro (Управление тест-кейсами) |
| `04-login-page.png` | Страница входа (Форма авторизации) |
| `05-registration.png` | Создание аккаунта (TC001: Регистрация пользователя) |
| `06-onboarding.png` | Настройка аккаунта (Onboarding при регистрации) |
| `07-rdi-setup.png` | Индивидуальный расчёт калорий (RDI — рекомендуемая норма) |
| `08-authorization.png` | Авторизация пользователя (TC002: Вход в систему) |
| `09-devtools.png` | Работа с DevTools (Проверка производительности) |
| `10-food-diary.png` | Дневник питания (TC003, TC004: Добавление продуктов) |

## Как добавить изображения

1. Сохраните скриншоты с указанными выше именами
2. Скопируйте их в папку `images/`
3. Сделайте commit и push в GitHub

```bash
git add images/
git commit -m "Add screenshots"
git push
```
