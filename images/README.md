# Скриншоты для приложения

Добавьте скриншоты тестирования в эту папку с следующими именами:

| Файл | Описание |
|------|----------|
| `01-registration.png` | Страница регистрации |
| `02-onboarding.png` | Настройка профиля |
| `03-rdi.png` | Расчёт калорий (RDI) |
| `04-signin.png` | Страница авторизации |
| `05-food-diary.png` | Дневник питания |
| `06-dashboard.png` | Главная страница (Dashboard) |
| `07-profile.png` | Профиль пользователя |
| `08-web-vitals.png` | Core Web Vitals (DevTools) |
| `09-kanban-miro.png` | Kanban-доска в Miro |
| `10-homepage.png` | Главная страница FatSecret |

## Как добавить изображения

1. Сохраните скриншоты с указанными выше именами
2. Скопируйте их в папку `images/`
3. Сделайте commit и push в GitHub

```bash
git add images/
git commit -m "Add screenshots"
git push
```
