## Тест-кейс 2 - поиск рецептов по продукту (позитивный)

**Назначение:** необходим для проверки результатов выдачи рецептов исходя из поисковой строки (первые 10).

**Ожидаемый результат:** на странице выдачи результатов отображается 10 рецептов, в названии которых содержится указанная при поиске подстрока.

**Ресурсы:**
- Поисковая строка: "tomato"

## Тест-кейс 3 - тестирование подсчёта употреблённых калорий

**Назначение:** отображение значения калорий, которые были употреблены исходя из добавленных записей продуктов.

**Ожидаемый результат:** на странице отчета за день калории отображаются как сумма записей употреблённых продуктов.

**Ресурсы:**
- Продукт: strawberries

## Тест-кейс 4 - добавление записей в личный дневник (журнал)

**Назначение:** на странице личного дневника должны отображаться все добавленные записи со станицы добавления.

**Ожидаемый результат:** добавленная запись отображается на странице.

**Ресурсы:**
- Текст для записи: "New record from test"
